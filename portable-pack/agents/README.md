# agents/ — the roster has two axes

**Tiers** (sonnet-low → opus-low → opus-med → opus-high) select CAPABILITY
for a spec: pick by spec-completeness and epistemic stakes (upskilling 09
rule 2, 06 rule 6).

**Named roles** (`intuition.md`) select a STANDING STANCE: read-only, a
different model family, an adversarial or advisory default — fences and
posture pre-baked into the definition instead of re-typed into every
prompt. The two axes compose in a wave: tier executors build, a babysitter
pass integrates, the sounding board vets the plan first. Full treatment:
upskilling [06-delegation.md](../upskilling/06-delegation.md) §"The second
axis".

Rules of the room: every executor follows `_report-contract.md`; the
sounding board must be a genuinely different model family (its disagreement
is the product); and keep the roster small — each definition is standing
cognitive surface, so retire roles that go unused.
